Aide

Pifomètre

(Page d'Interconnexion Fantoir-OSM pour Minimiser l'Enorme Tas des Rues Egarées)

1. Sélection de la commune. Sélection de la commune par code Insee. Pour les codes des communes en Corse, 'a' et 'b' sont compris (exemple: Ajaccio -> 2A004).

2. Date du dernier rapprochement entre les données OpenStreetMap et Fantoir au sein de BANO.

Les boutons

3. Communes voisines. Affiche des liens vers les pages Pifomètre des communes environnantes.

4. Mise à jour. Au survol du bouton, une infobulle indique l'heure des dernières données OSM permettant de mettre à jour BANO.
Cliquer sur le bouton force la mise à jour de BANO sur la commune. Par défaut, une mise à jour est faite toutes les nuits.

5. Export CSV. Télécharge un fichier CSV avec tout le tableau Pifomètre de la commune affichée.

6. Les filtres

Voies et lieux-dits Pour choisir d'afficher uniquement les voies, ou uniquement les lieux-dits, ou les deux.

Rapprochement
• Le filtre vert permet d'afficher ou non les voies/lieux-dits provenant de Fantoir, de la BAN ou du Cadastre et ayant trouvé une correspondance (par leur nom ou leur code Fantoir) avec un endroit dans OSM.
• Le filtre orange permet d'afficher ou non les voies/lieux-dits provenant de Fantoir, de la BAN ou du Cadastre et n'ayant pas trouvé d'endroit correspondant dans OSM.

Filtre bleu Apparaissent ici les lieux «trouvés dans OSM et inconnus de Fantoir». Cette liste remonte les noms d'endroits (rues, lieux-dits, cimetières, écoles, parkings...) qui pourraient avoir une correspondance dans Fantoir mais qui n'en ont pas trouvé – soit parce qu'il n'y a rien de correspondant dans le Fantoir de la commune, soit parce qu'il y a une erreur à identifier et corriger dans OSM (orthographe...).

Adresses
• Le filtre rose permet d'afficher ou non les voies/lieux-dits ayant des adresses référencées dans la BAN.
• Le filtre gris permet d'afficher ou non les voies/lieux-dits n'ayant pas d'adresses référencées dans la BAN.

Le tableau

7. Code Fantoir. Code de la voie ou du lieu-dit trouvé dans le ficher Fantoir de la commune. Le code comporte 9 caractères pour préparer le format du futur fichier Topo, successeur du fichier Fantoir encore non publié à ce jour (août 2023).

8. Création Fantoir. Date de création du code Fantoir (1 janvier 1987 pour les plus anciens).

9. Libellé Fantoir. Nom lié au code Fantoir selon le fichier Fantoir de la commune.

10. Libellé OSM. Nom trouvé dans OpenStreetMap. Il peut provenir d'une voie (highway), d'un lieu-dit (place) ou d'une adresse (addr:street ou addr:place).
Si le nom OSM correspond à un nom de voie ou lieu-dit trouvé dans Fantoir, il est accompagné d'une pastille ronde et verte : on dit qu'il est «rapproché».
Si aucun nom dans OSM ne correspond à la voie ou au lieu-dit trouvé dans Fantoir, s'affiche juste une pastille orange en forme de losange.

11. Libellé BAN ou Cadastre. Nom trouvé dans la BAN pour les voies, ou dans le Cadastre pour les lieux-dits. La source est précisée sur chaque ligne avec un pictogramme BAN ou Cadastre.

12. Carte. Lien vers la carte Openstreetmap.org centrée sur la voie ou le lieu-dit.

13. Édition. Liens ouvrant l'éditeur en ligne iD centré sur la voie ou le lieu-dit, ou le chargeant dans JOSM s'il est déjà ouvert.

14. Statut Fantoir. Liste des «anomalies» que peut comporter une voie ou un lieu-dit venant de Fantoir. Statut attribué à chaque voie ou lieu-dit disposant d'un code Fantoir, pour indiquer un éventuel problème empêchant ou non le rapprochement avec OSM.

15. Intégrer les adresses. Liens permettant d'intégrer dans OpenStreetMap les adresses affectées à cette voie ou ce lieu-dit selon la BAN. Le nombre affiché correspond au nombre de numéros d'adresses recensés par la BAN et non encore présents dans OSM.
Cliquer sur le lien ouvre un nouveau calque sur JOSM, comportant lesdites adresses soit sous forme de points (addr:housenumber et addr:street ou addr:place) soit par relation associatedStreet.
Si toutes les adresses recensées par la BAN sont déjà existantes dans OSM, cette colonne affiche «✔ Adresses intégrées».

16. Qualifier les adresses. Liens vers la page de qualification des adresses. Cette page affiche une carte de la voie ou du lieu-dit, avec les adresses déjà présentes dans OSM, les adresses venant de la BAN pas encore intégrées dans OSM, et un tableau listant les adresses et leurs anomalies éventuelles (sur le même modèle que les «statuts Fantoir».
